E-Commerce & Retail

Customer Churn & Retention Optimization Analysis

Business Problem: E-commerce application experiencing an unmeasured drop in quarter-over-quarter user retention.
Dataset: 12-month anonymized transactional and user behavior relational database logs.
Tools Used: Python (Pandas, Seaborn), SQL (PostgreSQL), Power BI.

Data Cleaning Process:

Handled null records within critical temporal tags using forward-fill strategy; filtered out system anomalies and duplicate transactional UUIDs; cast datatype structures to optimal memory configurations.

Key Performance Indicators (KPIs):

  • Customer Churn Rate
  • Customer Lifetime Value (CLV)
  • Cohort Retention Rate

Key Insights:

Analysis revealed that user drop-offs spikes significantly by 34% if the first order delivery experiences a delay over 48 hours, particularly within suburban distribution centers.

Recommendations:

Implement real-time tracking checkpoints for transit lines serving vulnerable regional hubs and run targeted automated promotional workflows for customers experiencing initial delivery lags.

Corporate Finance

Regional Sales Revenue Performance Dashboard

Business Problem: Leadership lacked unified visibility into decentralized multi-regional product line margins.
Dataset: Standardized flat-file ERP extractions across multiple localized operating subsidiaries.
Tools Used: Power Query, Power BI, DAX, Excel.

Data Cleaning Process:

Consolidated country-level regional CSV sheets via Power Query; standardizing multi-currency sales entries into uniform USD baselines; validated country taxonomy naming conventions to avoid regional cross-reporting mapping flaws.

Key Performance Indicators (KPIs):

  • Gross Profit Margin
  • Year-over-Year (YoY) Growth
  • Sales Target Variance

Key Insights:

Two key legacy hardware product lines accounted for 65% of revenue but operated under tightening margins (<12%), while high-margin software service add-ons remained heavily underrepresented in sales workflows.

Recommendations:

Pivot cross-selling incentives to favor software extensions; run localized operational cost auditing in bottom-performing regions to scale back high-overhead logistics variables.
